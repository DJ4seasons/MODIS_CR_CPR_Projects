# MODIS_CR_CPR_Projects
Codes used for MODIS Cloud Regime and/or Cloud-Precipitation Regime-based projects

## 1 Basic codes for CR/CPR
CR and CPR data at original daily and 1-degree resolution are available at https://zenodo.org/records/18356023. This is the Version of "Upto_2024.12_wIMERGv07B."
Codes in this folder show the pattern of centroids and mean relative-frequency-of-occurrence (RFO) maps, etc.

## 2 Codes for monthly CR-groups
Monthly CR-groups are used in the paper, https://doi.org/10.1029/2025JD045207, and data is available at https://zenodo.org/records/17831744. This is the version of "v2002.09-2024.08_on12052025." Codes in this folder show how to utilize monthly CR-group data.

## 3 Codes for monthly low-CR prediction from cloud-controlling-factors (CCFs)
This is based on the paper (submitted). Codes in this folder show details of this work, including sampling strategy, neural network model training, and evaluation figures, etc.
